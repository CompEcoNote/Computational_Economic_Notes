## Submission Guidelines

### What We Publish
- Research notes (max 2500 words)

### Requirements
- Open source code (GitHub repository)
- Reproducible computational results
- Clear documentation
- MIT or similar open license

### Review Process
1. **Initial screening** (1 week)
2. **Peer review** (2-4 weeks)
3. **Revision period** 
4. **Final acceptance**
